# Employee-Management-System
Employee Management System is a web-based application developed to efficiently manage employee records within an organization. It allows administrators to perform essential operations such as adding, updating, viewing, searching, and deleting employee information through a simple and user-friendly interface. The system helps reduce manual paperwork, improves data accuracy, and enables quick access to employee details.

# Features
-> Add new employee record 
-> View employee details
-> Update employee information
-> Delete employee records
-> Search employees by ID or name
-> User-friendly interface
